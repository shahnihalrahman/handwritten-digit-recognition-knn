# Handwritten Digit Recognition using KNN

## Overview
This project implements a Handwritten Digit Recognition System using the K-Nearest Neighbors (KNN) Machine Learning algorithm.

The system is trained on handwritten digit datasets and predicts numeric digits based on image input.

---

## Features
- Handwritten digit classification
- K-Nearest Neighbors (KNN) algorithm
- Image preprocessing
- Accuracy evaluation
- Data visualization
- MNIST dataset support
